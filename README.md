## Course Overview

This repository contains all the resources, code samples, and materials for the Udemy course titled "Spring Full Course." In this course, you will learn the following topics:

- Spring Core
- Spring MVC
- Databases and JDBC API
- Hibernate and JPA
- Spring Boot
- Spring Security
- Spring REST
- JWT Authentication
